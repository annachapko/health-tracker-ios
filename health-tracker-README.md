# Health Tracker iOS App

An iOS app for tracking daily calorie intake and nutrition metrics. Built with Swift and SwiftUI as part of an iOS Development course at CUNY Hunter College.

---

## Features

- **Food logging** — search for food items and log them to your daily diary using the Nutritionix API
- **Nutrition metrics** — view calorie and nutrient breakdowns per meal and per day
- **Built-in calendar** — track intake day by day with a structured logging system
- **User accounts** — create a profile and log in to keep your data personal
- **Health tips** — browse wellness articles and health guidance
- **Tab navigation** — clean SwiftUI tab-based layout for easy browsing

## How to Run

1. Clone the repo
2. Open `Health App.xcodeproj` in Xcode
3. Select a simulator (iPhone 15 recommended)
4. Press **Run** (⌘R)

> Requires Xcode 15+ and iOS 17+
